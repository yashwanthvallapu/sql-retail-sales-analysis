# Retail Sales Analysis -SQL Project

## Project Overview
This Project focus on analyzing retail sales data using SQL to uncover meaningful insights such as total revenue,top-selling products,and customer purchase behavior.
the project demonstarte practical SQL skills used in real-world data analysis scenarios.

## Objective
* Analyze sales performance using SQL queries
* Identify top-selling products and high-value customers
* Understand city-wise and category-wise revenue trends
* Practice real-world SQL concepts used by data analytics.
  
## Tools and Technologies
* Database: MySQL
* Language: SQL
* Concepts Used:
  * Joins
  * Aggregations
  * Subqueries
  * CTEs
  * Views
  * Window Functions
    
## Database Schema
The database consists of the following tables:
* Customers - customer details
* Products - product information
* Orders - order header data
* order_items - order transaction details

## Key Analysis Performed
* Total and average sales calculation
* Top-selling products analysis
* Customer purchase ranking
* City-wise revenue contribution
* Month-wise sales trend analysis
